# **App Name**: Orin

## Core Features:

- User Authentication: Secure signup and login with Bolt Database Auth and role-based access control (student, mentor, admin).
- Profile Management: Create and manage student and mentor profiles with detailed information and expertise selection.
- Mentor Discovery: Browse and search for mentors based on categories, subjects, exams, price, rating, language, and experience.
- Booking System: Detailed mentor profiles with availability calendars, session booking, and payment integration.
- AI Mentor Recommendation: An AI tool to analyze student profiles and recommend the best-matching mentors based on goals and expertise.
- Real-time Chat: Enable communication between students and mentors with a real-time chat interface.
- Admin Dashboard: Admin dashboard for mentor approvals, user management, category management, and revenue analytics.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to evoke trust, reliability, and professionalism, fitting for an educational platform.
- Background color: Light gray (#F0F0F0) to maintain a clean, uncluttered layout that emphasizes content.
- Accent color: Golden yellow (#FFD700) for highlighting key interactive elements such as buttons, links, and notification badges to add warmth and improve the user experience.
- Headline font: 'Space Grotesk', a sans-serif font, for a modern and clean aesthetic.
- Body font: 'Inter', a versatile sans-serif font that is readable for both screen and print.
- Use flat, minimalist icons for easy recognition.
- Card-based layouts with clear separation and categorization.
